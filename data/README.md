# Data Directory

## 1. Processed Data
- `processed/filtered_variants.csv`: SNP data after quality control.
- `processed/statistics_results.tsv`: Summary of population test results.

## 2. Simulated Data
- `simulated/simulated_population.txt`: Synthetic genetic data for testing ghost populations.

## 3. Metadata
- `metadata/sample_metadata.tsv`: Labels for individuals in the dataset.

## 4. Notes
- Large files are stored externally at [HPC storage](hpc-path).
- Raw data can be accessed via [Zenodo](link).
